== Description ==

Simply add a reCAPTCHA v2 or v3 to your WordPress registration form.

== Installation ==

1. Install the plugin from your WordPress admin, or upload and install the plugin folder to your plugins directory (e.g. /wp-content/plugins/)
2. Activate the plugin
3. Go to Just Recaptcha > Settings and configure the options
